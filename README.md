# Show Segment Timer

Professional timing tool for TV directors and producers to plan and track show segment timings.

## Features

- Add multiple segments with custom names and durations
- Real-time total runtime calculation
- Set target duration and see over/under time
- Drag-and-drop segment reordering
- Export segment list to clipboard
- Data saved locally in browser

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build

```bash
npm run build
```

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons
